# Welcome

이전에는 velog에서 간단하게 포스팅했지만, 아쉽게도 연재하지 못하였습니다.
<br>
[이전 블로그](https://velog.io/@goneki9713/posts)

이번에 새롭게 github io blog를 만들게 되면서 프로젝트도 새롭게 시작할 계획입니다.
<br>
HW, SW 양쪽 광범위하게 다루는게 목표입니다.

HW 내용으로는 주로 inferface에 대해 포스팅할 계획입니다.
<br>
이전에 custom DMA를 만든 프로젝트, 앞으로 진행할 SPI, I2C interface 구현 프로젝트를 올리는 것을 생각하고 있습니다.

SW 내용으로는 아마 운영체제 내용이 많이 올라갈 예정입니다.
추후 HW 플랫폼이 완성되면 FPGA에 petalinux를 올릴건데 이 때 아마 많은 내용들이 올라가지 않을까 싶습니다.

전체 그림은 PS-PL 통합 시스템이고 초기에는 baremetal 수준으로 진행할 예정입니다.
여기서는 DMA engine, SPI, I2C controller가 구성되어있고 다른 board와 통신하는 그림을 생각하고 있습니다.


나중에는 이걸 petalinux상에서 device driver를 작성해서 OS level에서 관리, 제어하는 방향으로 진행할 계획입니다.

이렇게 하면 HW, SW에 대한 전반적인 내용을 자연스럽게 프로젝트를 진행하면서 다룰 수 있을 것입니다.

그리고 여기서 branch를 만들어서 code 관리를 위한 git 사용, shell 사용... 등 기타 알고 있으면 편한 것들에 대해서도 언급할 예정입니다.
